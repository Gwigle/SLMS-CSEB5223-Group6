## Project Overview
The Student Learning Management System (SLMS) is a console-based Java application designed to manage students, lecturers, courses, assignments and learning materials in a centralized system.
This project applies object-oriented programming principles, modular design and continuous integration practices.

## Coding Standards
- 4 spaces indentation
- PascalCase for class names
- camelCase for methods and variables
- UPPER_CASE for constants
- One class per file
- Maximum 120 characters per line

---

## Continuous Integration Rules

- Main branch is protected
- No direct commits to main
- Feature branches must be created
- Pull Request required before merging
- At least one reviewer approval required

Commit message format:
[Module] Short Description

Example:
[Course] Added course enrollment validation
